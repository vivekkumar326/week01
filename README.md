# week01
Python Basics project demonstrating variables, user input, and formatted output through a personal introduction program.

This project is a simple Python program created as part of the Arena Develop Virtual Internship – Week 1 Task.
The program collects basic information from the user such as name, age, city, and hobby, and then displays a personalized welcome message.
